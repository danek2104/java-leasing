# Leasing System

Система управления лизингом автомобилей. Приложение позволяет управлять пользователями, клиентами, автомобилями, договорами лизинга и заявками.

## Требования

Для запуска проекта вам понадобятся:

*   Java JDK 17 или новее
*   Maven
*   PostgreSQL

## Настройка базы данных

Перед запуском приложения необходимо создать базу данных PostgreSQL.

1.  Убедитесь, что PostgreSQL запущен.
2.  Создайте базу данных с именем `leasing_db`.

Настройки подключения по умолчанию (можно изменить в `src/main/resources/application.properties`):
*   **URL:** `jdbc:postgresql://localhost:5432/leasing_db`
*   **Username:** `postgres`
*   **Password:** `postgres`

## Запуск приложения

### Через командную строку (Maven)

1.  Откройте терминал в корневой папке проекта.
2.  Выполните команду для запуска:

    ```bash
    ./mvnw spring-boot:run
    ```
    (или `mvn spring-boot:run`, если Maven установлен глобально)

### Сборка и запуск JAR

1.  Соберите проект:
    ```bash
    ./mvnw clean package
    ```
2.  Запустите полученный jar-файл:
    ```bash
    java -jar target/leasing-system-0.0.1-SNAPSHOT.jar
    ```
    *(Примечание: имя jar-файла может отличаться в зависимости от версии в pom.xml)*

## Доступ к приложению

После успешного запуска приложение будет доступно по адресу:
[http://localhost:8080](http://localhost:8080)

## Тестовые учетные записи

При первом запуске автоматически создаются следующие пользователи (см. `DataInitializer.java`):

| Роль | Логин | Пароль |
| :--- | :--- | :--- |
| **Администратор** | `admin` | `admin` |
| **Менеджер** | `manager` | `manager` |
| **Клиент** | `client` | `client` |

## Функционал

*   **Администратор:** Управление пользователями и ролями.
*   **Менеджер:** Управление клиентами, автомобилями, просмотр и одобрение заявок, создание договоров.
*   **Клиент:** Просмотр доступных автомобилей, подача заявок на лизинг, просмотр своих договоров.
